Recording

There are 6 word lists. Each list should take around 30 seconds.

Example of how it should sound. Note the speed, flat intonation, and spaces between words.

Instructions

Click the arrow next to a step for further technical instructions.
  1. Download 'Audio Recorder' and change the settings
    • File format: wav
    • Channels: mono
    • Sampling rate: 44100 Hz
    • Bit depth: 16 bit
  2. Hold the phone such that the charging port is 3-4 inches away from your face.
        DO NOT USE HEADPHONES OR A MIC! JUST YOUR PHONE!
  3. Record a WAV file for each list named 1.wav, 2.wav, etc.
  4. Export the 6 files to the device you're taking this survey on.

    1. Click "edit" in the upper right
    2. Select all your recordings
    3. Click the export button in the bottom left

    Google Drive, Email, Airdrop, whatever method you would normally use to move files to a computer.

  5. Compress your files into a zip named wavs.zip
    • MacOS
      1. Open Finder and go to your wav files
      2. Select your 6 files
        • Select multiple files in any order by holding Command and clicking the files you want.
        • Select a range of files by holding shift and clicking the first and last files in the range.
      3. Right-click and choose "Compress"
      4. A .zip file will be created
      5. Name it wavs.zip
        1. Right-click the zip file
        2. Choose "Rename"
        3. Type "wavs.zip"
    • Windows
      1. Open File Explorer and go to your wav files
      2. Select your 6 files
        • Select multiple files in any order by holding Ctrl and clicking the files you want.
        • Select a range of files by holding Shift and clicking the first and last files in the range.
      3. Right-click and choose "Send to" > "Compressed (zipped) folder"
      4. A new zip folder will be created
      5. Name it wavs.zip
        1. Click once on the zip folder to select it
        2. Press F2 (or right-click and choose "Rename")
        3. Type "wavs.zip"
